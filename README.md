# 摄影策划集

摄影风格策划与色彩方案管理工具，支持 12 种摄影风格配色：日系清新、法式慵懒、中式古风、胶片复古、都市冷感、情绪暗调、多巴胺、森系少女、强闪光、美式街拍、糖水片、私房。

## 功能

- 12 种摄影风格一键切换
- 纯静态单文件，无需服务器
- 响应式布局，手机/电脑均可使用

## 技术栈

HTML + CSS + JavaScript（单文件纯静态页面）

## 手机端访问

[https://foreverjg.github.io/photography-plan/](https://foreverjg.github.io/photography-plan/)

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 部署更新

```bash
./deploy.sh
```

或手动执行：

```bash
git add index.html && git commit -m "更新" && git push
```

推送后 GitHub Pages 自动部署，约 30 秒生效。
