# 摄影策划集

摄影风格策划与色彩方案管理工具，支持 12 种风格一键切换。

## 12 种风格

日系清新 · 法式慵懒 · 中式古风 · 胶片复古 · 都市冷感 · 情绪暗调 · 多巴胺 · 森系少女 · 强闪光 · 美式街拍 · 糖水片 · 私房

## 手机访问

[https://foreverjg.github.io/photography-plan/](https://foreverjg.github.io/photography-plan/)

## 本地使用

浏览器打开 `index.html`

## 部署

**Windows** — 双击 `deploy.bat`

**Git Bash** — `./deploy.sh`

推送后 GitHub Pages 约 30 秒自动更新。

## 技术栈

HTML + CSS + JavaScript（纯静态单文件）
